Damage restoration of ancient Indian wall paintings using Deep Learning

Under Dr. Dhiraj Sangwan

Worked on identifying damaged regions from damaged ancient Indian murals and using image inpainting techniques to digitally restore them. We initially created an exhaustive dataset by synthetically damaging clean images of wall paintings using binary masks and textures, followed by using segmentation models for damage segmentation and GAN architecture for image inpainting.
